# edunomics-intern-work

### NodeJs-Backend

### `npm install`

It install all the dependencies needed for the project to work.

### `node server.js` or 
### `nodemon server.js` 

Starts the server at port 5000.<br />
Open [http://localhost:5000](http://localhost:5000) to view it in the browser.
